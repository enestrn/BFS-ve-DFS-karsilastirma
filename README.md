# Yapay Zeka Uzman Sistemler: Temizlik Ajanları Simülasyonu

## Türkçe 
Bu proje, **Yapay Zeka Uzman Sistemler** dersi kapsamında, iki odadan oluşan bir ortamda temizlik yapması gereken farklı ajan modellerinin simülasyonunu içermektedir. Proje, **BFS** (Breadth-First Search) ve **DFS** (Depth-First Search) gibi arama algoritmalarının yanı sıra, her ajanın (Rastgele Ajan, Tablo Tabanlı Ajan, Refleks Ajan ve Model Tabanlı Ajan) çevresel durumu algılayıp doğru aksiyonlar alarak temizliği sağlama sürecini simüle etmektedir. Her ajan modelinin başarı oranları karşılaştırılarak, her durumda hangi modelin daha etkili olduğu analiz edilmiştir. Bu proje, **yapay zeka ajanları** ve **karar verme mekanizmaları** üzerine yapılan bir çalışmadır.

## English
This project involves the simulation of different agent models that are tasked with cleaning two rooms in an environment, as part of the **Artificial Intelligence Expert Systems** course. The project uses search algorithms like **BFS** (Breadth-First Search) and **DFS** (Depth-First Search), along with various agent types (Random Agent, Table-Based Agent, Reflex Agent, and Model-Based Agent), to simulate the process of detecting the environment's state and taking appropriate actions to clean. The success rates of each agent model are compared, and the most effective model for each scenario is analyzed. This project is focused on **artificial intelligence agents** and **decision-making mechanisms**.
